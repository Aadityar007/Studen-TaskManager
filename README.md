# TaskMaster App

A simple **Task Management application** built using **React + TypeScript + Vite**.  
Users can add tasks, mark them as completed, and filter tasks by status.  

---

## Features

- Add new tasks
- Mark tasks as completed
- Filter tasks by status (Pending / Completed)
- Responsive and clean UI

---

## Run Locally

**Prerequisites:** Node.js

1. Clone the repository:  
   ```bash
   git clone https://github.com/Aadityar007/Studen-TaskManager
   cd Studen-TaskManager
 2.  Install dependencies:
    npm install
3. Start the development server:
    npm run dev
